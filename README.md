GitHub Profile Finder is a simple and responsive website that allows users to search for any GitHub username and instantly view the user’s profile details along with their public repositories. # 🚀 GitHub Profile Finder

GitHub Profile Finder is a simple and responsive web application that allows users to search for any GitHub username and view their public profile details along with repositories.

---

## 📌 Features

- 🔍 Search GitHub users by username  
- 👤 Display user profile details:
  - Profile picture
  - Name & bio
  - Followers, following, and repository count  
- 📦 View public repositories
- ⭐ Sort repositories by stars
- 📱 Fully responsive design
- 🎨 Clean and beginner-friendly UI

---

## 🛠️ Technologies Used

- **HTML** – Structure of the website  
- **CSS** – Styling and responsiveness  
- **JavaScript** – Logic and API handling  
- **GitHub REST API** – Fetching user and repository data  

---


---

## ⚙️ Working of the Project

1. User enters a valid GitHub username  
2. On clicking **Search**, data is fetched from the GitHub API  
3. User profile details are displayed  
4. Public repositories appear below the profile  
5. Repositories can be sorted by stars for better readability  

---

## 🚀 How to Run the Project

1. Clone or download the repository  
2. Open `index.html` in any modern browser  
3. Enter a GitHub username to view details  

_No additional setup required._
